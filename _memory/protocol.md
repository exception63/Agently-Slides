# Protocol: Hermes-style Project Memory

## Read Order
1. `active.md`
2. `episodic_index.md` for “what happened/where is it”
3. `decisions.md` for “why”
4. `procedures/` for reusable workflows
5. `archive/` only when needed

## Forgetting
Forgetting means demotion, not deletion. Keep active small. Move completed detail to timeline, episodic_index, procedures, or archive.

## Active Budget
Target: under 900 Chinese characters. Hard limit: 1,500 Chinese characters.

## Memory Flush
After meaningful work, update active, decisions, timeline, episodic_index, and procedures only as needed.
